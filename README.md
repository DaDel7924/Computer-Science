# 总述
408科目要求部分题目通过代码解题，为了在各设备上观看方便，于此发行本仓库。
本仓库中的代码基于C/C++语言完成，2013年以前允许使用Java，但目前已不适用。

# 关于408
众所周知，408科目中具有4个分支：DS，OS，Net和计算机组成。其中对于DS和OS具有代码的要求。
